Class Autoloader

The Autoloader eliminates the need to require or include every individual PHP file to be used.

Note: This Autoloader assumes a valid PSR-4 directory structure. See the PHP FIG documentation regarding PSR-4 standards for more information. For more information about autoloading in PHP, see the PHP.net page on the topic.

# addPrefix( string $prefix, string $dir )

Adds a prefix or namespace for the autoloader to check when looking for a class.

Adds a prefix or namespace for the autoloader to check when looking for a class.

A prefix can either be a first-level namespace or the first term of a class name that uses underscores to indicate its directory structure (such as the Zend Framework).

Example:

$autoloader->addPrefix('MyNamespace', '/some/path/here');

// Both of the following will be looked for in the path above. The Autoloader will look for
// the file "/some/path/here/MyNamespace/SomeClass.php"
$class1 = new \MyNamespace\SomeClass();
$class2 = new MyNamespace_SomeClass();
